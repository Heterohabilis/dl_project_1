# DL Project 1

This is the repository of the 1st project of deep learning.

## Startup

- Set the environment:
  
  ```bash
  # create new virtual env. skip if already have one
  python -m venv .venv

  # activate the virtual env
  source .venv/bin/activate

  # install all required packages (run once)
  pip install -e .[dev]
  ```
